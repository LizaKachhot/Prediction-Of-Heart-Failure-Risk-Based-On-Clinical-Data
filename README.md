# ❤️ Prediction of Heart Failure Risk Based on Clinical Data

*Smarter Insights for a Healthier Heart*

---

## Overview
This project focuses on predicting the risk of heart failure using machine learning techniques applied to clinical data.
The system analyzes patient health parameters and provides risk prediction along with explanations and preventive recommendations.

---

## Features
- Heart failure risk prediction using machine learning
- Interactive web application built with Streamlit
- Explainable risk factors and recommendations
- Outlier detection and feature scaling
- Downloadable PDF medical report

---

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- XGBoost
- Streamlit
- Plotly, Matplotlib
- FPDF

---
